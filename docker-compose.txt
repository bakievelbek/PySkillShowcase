Docker Compose - это инструмент для определения и запуска многоконтейнерных приложений Docker с использованием
YAML-файла для конфигурации сервисов приложения. Это позволяет пользователям легко настраивать связанные контейнерные
приложения и управлять ими как одним целым. С помощью Docker Compose вы можете определить настройки контейнеров,
сетей и томов в одном файле, а затем с помощью одной команды создать и запустить все определенные сервисы. Это очень
удобно для разработки, тестирования и развертывания сложных приложений, состоящих из множества контейнеров, которые
должны работать вместе.

Volume в контексте Docker - это механизм для сохранения или обмена данными между контейнерами и хостовой системой.
Volume позволяет сохранять данные вне жизненного цикла контейнера, что означает, что данные не удаляются вместе с
удалением контейнера, и обеспечивает возможность их постоянного хранения. Также volume можно использовать для обмена
данными между разными контейнерами. Volumes поддерживаются драйверами, которые могут предоставлять дополнительные
функции, такие как шифрование, сжатие или удаленное хранение. Создание и управление volumes осуществляется с помощью
команд Docker CLI или через Docker Compose файлы.
